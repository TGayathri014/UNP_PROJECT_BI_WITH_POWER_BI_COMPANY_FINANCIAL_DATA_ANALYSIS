# UNP_PROJECT_BI_WITH_POWER_BI_COMPANY_FINANCIAL_DATA_ANALYSIS

I completed power bi project on company financial data set in collaboration with unp education.
This project involved analyzing key financial metrics such as revenue, profit, expenses, and growth trends using interactive dashboards.
I worked with a real-world dataset to apply data cleaning, transformation, and visualization techniques in Power BI.
The project enhanced my understanding of financial performance analysis and data-driven decision-making.
It also helped me develop skills in creating dynamic reports, KPIs, and comparative visual insights.
Guidance and structured learning support were provided by UNP Education throughout the project.
Overall, the experience strengthened my analytical, technical, and teamwork skills while working on a practical business-oriented financial dataset.
